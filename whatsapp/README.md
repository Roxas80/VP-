# WhatsApp Business — Настройка

## Быстрый старт без API

### 1. Установка
- Скачать WhatsApp Business (отдельное приложение)
- Зарегистрировать номер кондитерской

### 2. Настройка профиля
- **Название:** vkusno_polezno_astana
- **Описание:** ПП торты, десерты, выпечка без сахара и глютена | Астана
- **Адрес:** (указать)
- **Часы работы:** (указать)
- **Каталог:** Добавить 3-5 товаров с фото

### 3. Быстрые ответы (шаблоны)

Создать через Настройки → Бизнес-инструменты → Быстрые ответы

| Команда | Сообщение |
|---------|-----------|
| `!привет` | Здравствуйте! 🍰 Мы — vkusno_polezno_astana. Полезные десерты без сахара и глютена в Астане. Что хотите заказать? |
| `!меню` | Наше меню:
1. Торт "Наполеон" ПП — 4500₸/кг
2. Чизкейк манго — 3000₸
3. Хлебцы без глютена — 1500₸

Подробнее: [ссылка на каталог] |
| `!заказ` | Для оформления заказа укажите:
- Что хотите
- Адрес доставки
- Телефон
- Удобное время |
| `!доставка` | Доставка:
- Район [уточнить]: бесплатно от 5000₸
- Другие районы: [уточнить стоимость]
- Самовывоз: [адрес] |
| `!спасибо` | Спасибо за заказ! 🙏 Подтверждаем: [повторить заказ]. Доставка [время]. Если вопросы — пишите! |

### 4. Ярлыки для организации
- Новый заказ
- В обработке
- Доставка
- Выполнен
- Отменён

### 5. Ссылка для клиентов
Создать короткую ссылку wa.me:
`https://wa.me/[номер]?text=Здравствуйте!%20Хочу%20заказать%20десерт`

## Переход на WhatsApp Business API (после успеха MVP)

### Требования
- Верификация бизнеса в Meta
- Business Manager
- Провайдер (Twilio, 360dialog, WATI)

### Стоимость
- Верификация: бесплатно (2-4 недели)
- Сообщения: ~$0.005-0.05 за штуку
- Провайдер: $20-100/мес

## Интеграция с Instagram

В профиле Instagram:
- Добавить кнопку "Связаться" → WhatsApp
- В stories делать свайп-ап на чат
