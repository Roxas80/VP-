# Telegram Bot — Продажи vkusno_polezno_astana

## Описание
Автоматизированный бот для приёма заказов кондитерских изделий.

## Функционал

### Основные команды
- `/start` — приветствие и меню
- `/catalog` — показать каталог
- `/cart` — показать корзину
- `/order` — оформить заказ
- `/help` — помощь

### Сценарий диалога
1. Пользователь запускает бота
2. Видит приветствие + кнопки: "Каталог", "Корзина", "Оформить заказ"
3. Выбирает товар → добавляет в корзину
4. Повторяет или переходит к оформлению
5. Указывает: адрес, телефон, время доставки
6. Подтверждение заказа
7. Уведомление оператору

## Техническая реализация

### Стек
- Python 3.9+
- python-telegram-bot 20+
- SQLite для хранения

### Структура файлов
```
telegram_bot/
├── bot.py              # Основной файл бота
├── config.py           # Конфигурация (токен, настройки)
├── handlers/
│   ├── start.py        # Обработчик /start
│   ├── catalog.py      # Каталог товаров
│   ├── cart.py         # Корзина
│   └── order.py        # Оформление заказа
├── models/
│   └── database.py     # Работа с БД
├── data/
│   ├── products.json   # Каталог товаров
│   └── orders.db       # База заказов
└── requirements.txt    # Зависимости
```

## Установка

```bash
pip install -r requirements.txt
python bot.py
```

## Настройка

1. Создать бота через @BotFather
2. Получить токен
3. Заполнить `config.py`
4. Заполнить `data/products.json` каталогом
