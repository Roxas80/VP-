## 6. Автоматическая проверка (по желанию)

1. Загрузите файл **`docs/plesk-check.php`** в корень сайта (туда же, где будет лежать `index.html`).
2. Откройте в браузере: `https://ваш-домен.ru/plesk-check.php`
3. Скопируйте весь текст со страницы и сохраните (или пришлите разработчику).
4. После этого **удалите** `plesk-check.php` с сервера.
