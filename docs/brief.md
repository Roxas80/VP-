# Бриф проекта: Продажи кондитерских изделий vkusno_polezno_astana

## Цель проекта
Запустить продажи кондитерских изделий через мессенджеры (Telegram + WhatsApp) с минимальными затратами и быстрой проверкой гипотезы.

## Описание бизнеса
- **Название:** vkusno_polezno_astana
- **Локация:** Астана, Казахстан
- **Продукция:** ПП торты, десерты, выпечка, хлеб без сахара и глютена
- **Instagram:** @vkusno_polezno_astana

## Целевые каналы
1. Telegram Bot — для автоматизации заказов
2. WhatsApp Business — для быстрого старта и обработки заказов

## MVP тест (24-48 часов)
### Критерий успеха
- 10 реальных заказов с оплатой
- Понятная конверсия: посетители → заказы
- Средний чек и частота повторных покупок

### Что включает MVP
- Каталог из 3-5 хитовых позиций
- Приём заказов через Telegram и WhatsApp
- Оплата: наличные/перевод при получении (не интегрировать сразу)
- Доставка: уточнить при заказе

## Данные для заполнения
- [ ] Названия и цены 3-5 хитовых товаров
- [ ] Фото товаров
- [ ] Контактный номер для приёма заказов
- [ ] Условия доставки (районы, стоимость, сроки)
- [ ] Способы оплаты

## Архитектура
Hub-and-Spoke:
- CEO → Бро (оркестратор) → Telegram Bot / WhatsApp
