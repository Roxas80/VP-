# Требования проекта vkusno_polezno_astana

## Функциональные требования

### Telegram Bot
- [ ] Приветственное сообщение с описанием ассортимента
- [ ] Каталог товаров с фото, описанием и ценами
- [ ] Добавление в корзину
- [ ] Оформление заказа (адрес, телефон, комментарий)
- [ ] Уведомление оператору о новом заказе
- [ ] Подтверждение заказа клиенту

### WhatsApp Business
- [ ] Быстрые ответы (шаблоны сообщений)
- [ ] Каталог товаров (ссылки или PDF)
- [ ] Обработка заказов вручную

## Нефункциональные требования
- Время отклика бота: < 3 секунд
- Доступность: 24/7 для приёма заказов
- Простота обновления каталога

## Ограничения
- Без интеграции платёжных систем на старте
- Без автоматической доставки (уточнение вручную)
- Тест только на 3-5 SKU

## Интеграции
- Telegram Bot API
- WhatsApp Business (не API)
- Instagram (источник трафика)

## Безопасность
- Не хранить данные карт клиентов
- Минимальная обработка персональных данных

## Масштабирование (после успеха MVP)
- Интеграция Kaspi/карт
- Автоматическая доставка
- CRM для клиентов
- Повторные заказы в один клик
