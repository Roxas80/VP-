### 3. Workflow для группы кондитеров:

**Роли:**
- Группа кондитеров: -5004785391
- Админ: @username (определяется)
- Кондитер: отвечает за приготовление
- Менеджер: подтверждает заказы

**Статусы заказа:**

1. **НОВЫЙ ЗАКАЗ** → Уведомление в группу
   - Кнопки: "✅ Подтвердить" | "❌ Отменить"
   - Тег: @менеджер

2. **ПОДТВЕРЖДЁН** → Уведомление "Начать готовку"
   - Кнопки: "👨‍🍳 Начать готовку"
   - Тег: @кондитер

3. **ГОТОВИТСЯ** → Статус обновляется
   - Кнопки: "✅ Готово к доставке"
   - Тег: @кондитер

4. **ГОТОВ К ВЫДАЧЕ** → Менеджер проверяет
   - Кнопки: "🚗 Назначить курьера" | "✅ Самовывоз"
   - Тег: @менеджер

5. **В ДОСТАВКЕ** → Курьер назначен
   - Уведомление клиенту
   - Ссылка на отслеживание

6. **ДОСТАВЛЕНО** → Клиент подтвердил
   - Архив
