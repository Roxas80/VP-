## 2. Архитектура сайта

### 2.1 Структура файлов

```
VPA/
├── index.html          # Единая страница, семантическая разметка
├── styles.css          # Все стили (переменные, блоки, адаптив)
├── script.js           # Плавная прокрутка, форма, WhatsApp
├── assets/             # Опционально: изображения, иконки
│   ├── images/
│   └── favicon.ico
├── vkusno_polezno.md   # Исходное ТЗ
└── ARCHITECTURE.md     # Этот документ
```

### 2.2 Технологический стек

- **HTML5** — семантика (header, main, section, footer), доступность (aria, alt).
- **CSS3** — CSS-переменные (палитра, шрифты), Flexbox/Grid, медиа-запросы.
- **Vanilla JS** — без фреймворков для быстрой загрузки и простоты хостинга.

Обоснование: минимальный стек, загрузка ≤ 2 сек, лёгкое масштабирование (при необходимости — перенос на React/Next.js).

### 2.3 Дизайн-система

**Цвета (CSS-переменные):**
- `--bg-main`: белый / молочный
- `--bg-cream`: кремовый, светлый беж
- `--accent-green`: мягкий зеленый
- `--accent-brown`: натуральный коричневый
- `--text-primary`, `--text-secondary`

**Типографика:**
- Заголовки: Playfair Display или DM Serif (Google Fonts).
- Текст: Inter или Manrope.

**Секции:**
1. **Hero** — полноэкранный блок, фото десерта, заголовок, 2 CTA (WhatsApp, «Смотреть меню»).
2. **О бренде** — текст + 4 преимущества (иконки/короткие тезисы).
3. **Продукция** — карточки (торты, десерты, хлеб, пироги): фото, название, описание, «Заказать».
4. **Галерея** — сетка 8–12 фото.
5. **Как заказать** — 3 шага (WhatsApp → Выбор → Получение).
6. **Отзывы** — 3–6 карточек отзывов.
7. **CTA** — заголовок «Готовы попробовать?» + кнопка «Заказать сейчас».
8. **Контакты** — WhatsApp, Instagram, TikTok, адрес, карта (iframe или ссылка на карты).

### 2.4 Функциональность

- **WhatsApp:** ссылка `https://wa.me/7XXXXXXXXXX?text=Здравствуйте,%20хочу%20заказать%20десерт` на всех кнопках «Заказать» / «Написать».
- **Форма заявки:** поля имя, телефон, комментарий; отправка — по желанию (Formspree, Telegram Bot, или только валидация + открытие WhatsApp с текстом).
- **Плавная прокрутка:** к якорям при нажатии «Смотреть меню» и к блоку контактов.
- **Адаптив:** брейкпоинты для мобильных (< 768px), планшетов (768–1024px), десктоп (> 1024px).

### 2.5 SEO

- Title: «Vkusno Polezno Astana — Полезные десерты в Астане»
- Meta description с ключами: полезные десерты Астана, торт Астана, ПП десерты Астана.
- Заголовки H1 (один) в Hero, H2 по секциям.
- Alt у всех изображений.

## 3. Порядок разработки

1. Базовая разметка `index.html` (все 8 блоков, якоря, форма).
2. Подключение шрифтов и базовая типографика.
3. Стили по блокам + CSS-переменные палитры.
4. Адаптив (мобильный первый подход).
5. `script.js`: плавная прокрутка, обработка формы, единая ссылка WhatsApp.
6. Подстановка реального номера WhatsApp и (при необходимости) адреса и карты.

После этого сайт готов к размещению на хостинге и к подключению аналитики (KPI: конверсия ≥ 5%, время на сайте ≥ 40 сек).
